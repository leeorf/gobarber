# :computer: Running the Frontend

### Requirements

- [GoBarber API](../backend) running

### First steps:
``` bash
# Go to frontend directory
$ cd frontend

# Install the dependencies
$ yarn

# Be sure the file 'src/services/api.ts' have the IP to your API

# Start the client
$ yarn start
```
